\frame{
A \textbf{characteristic function game} $G$ is given by a pair $(N,v)$ where $N$ is the number of players and $v:2^{[N]}\to\mathbb{R}$ is a \textbf{characteristic function} which maps every coalition of players to a payoff.
}

\frame{
$A$lice, $B$ob, $C$eline share a meal:
\pause
$$
v(c)=\begin{cases}
80,&\text{if }c=\{A\}\\
56,&\text{if }c=\{B\}\\
70,&\text{if }c=\{C\}\\
80,&\text{if }c=\{A,B\}\\
85,&\text{if }c=\{A,C\}\\
72,&\text{if }c=\{B,C\}\\
90,&\text{if }c=\{A,B,C\}\\
\end{cases}
$$
}

\frame{
